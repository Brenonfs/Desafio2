Implementar um sistema para cadastrar e gerenciar informações de várias escolas na rede.

Gerenciamento de Escolas:
Desenvolver funcionalidades para cadastrar escolas.

Gerenciamento de Professores:
Desenvolver funcionalidades para cadastrar professores em uma escola.

Gerenciamento de Turmas:
Desenvolver funcionalidades para cadastrar turmas em uma escola.

Gerenciamento de Grade Escolar:
Desenvolver funcionalidades para cadastrar disciplinas e turmas. Um professor não pode dar aula para mais de uma turma no mesmo horário.

Matrícula de Alunos:
Criar um sistema de matrícula para alunos em uma escola. O sistema deve permitir a matrícula de alunos em uma turma com vagas disponíveis.

Comunicação Escola-Aluno-Responsável:
Introduzir um sistema de comunicação que permita a troca de informações entre escola, alunos e responsáveis, como boletins, comunicados e eventos.



Informações adicionais:

Pense que voce vai desenvolver o sistema de gerenciamento do São José. Mas que o mesmo sistema vai ser aplicado em outras escolas. Então, o sistema deve ser genérico o suficiente para ser aplicado em outras escolas.

Será necessário criar um sistema de login para que cada escola tenha acesso apenas aos seus dados.
Os professores devem ter o seu login e senha para acessar o sistema para visualizar as turmas que lecionam e consultar sua grade de horário.
Os alunos devem ter o seu login e senha para acessar o sistema para visualizar as turmas que estão matriculados e consultar sua grade de horário.




FUNCIONALIDADE FUTURA:
Sistema de Notas e Avaliações:
Desenvolver funcionalidades de avaliações para lançamento de notas, cálculo de médias e geração de relatórios de desempenho acadêmico dos alunos.
